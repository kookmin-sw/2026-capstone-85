Overview

Accountit은 CPA 채용 정보를 한곳에 모으는 플랫폼입니다

수습 가능 여부, KICPA 조건, 마감일, 회사 정보를 함께 보여 주어 회계 직무 탐색 시간을 줄이는 것을 목표로 합니다.

01

CPA 공고 큐레이션

여러 채널의 회계 직무 공고를 모아 출처와 마지막 확인 시간을 함께 제공합니다.

02

조건 기반 탐색

실무수습, KICPA, 직무, 연차, 마감일 기준으로 필요한 공고만 좁혀 봅니다.

03

AI 지원 판단 보조

이력서와 공고 조건을 비교해 적합도와 보완 포인트를 요약합니다.

System Architecture

서비스 아키텍처

Next.js 웹, NestJS API, Prisma/PostgreSQL, S3 정적 자산으로 구성된 CPA 채용 큐레이션 서비스입니다.

Monorepo

  • apps/web
  • apps/api
  • packages/shared
  • prisma

Stack

  • Web: Next.js, TypeScript, Tailwind CSS
  • API: NestJS, Prisma ORM, PostgreSQL
  • Auth: argon2, HTTP-only JWT cookie
  • UI: shadcn/ui, lucide-react

Roles

  • JOB_SEEKER
  • COMPANY
  • ADMIN
