# Accountit

![Accountit hero](apps/web/public/landing/accountit-hero.png)

**Accountit**은 회계사, 회계사 준비생, 회계법인을 위한 **CPA 특화 채용 큐레이션 플랫폼**입니다.

흩어져 있는 회계사 채용 공고를 한곳에 모아 제공하고, KICPA 조건, 수습 가능 여부, 실무수습기관 인정 가능성, 마감일, 직무군, 회사 유형, 경력 조건처럼 CPA 구직자에게 중요한 기준으로 공고를 탐색할 수 있도록 돕습니다. 사용자는 공고 요약, AI 이력서 적합도 분석, 기업 정보 비교, 마감일 캘린더, 커뮤니티 기능을 통해 채용 과정을 더 체계적으로 관리할 수 있습니다.

## One-Line Pitch

회계사 채용 공고를 한 곳에 모아 **마감일, 수습 가능 여부, KICPA 조건, 직무군, 연차** 기준으로 정리해주는 **CPA 전용 채용 큐레이션 플랫폼**입니다.

| 사용자 | 니즈 |
| --- | --- |
| 회계사 및 CPA 자격 보유자 | 감사, 세무, FAS, Deal, 내부회계, 인하우스 등 다양한 회계 직무로 이직하거나 커리어를 확장하고 싶습니다. |

## Problem

회계사 채용 공고는 여러 채널에 흩어져 있어 지원자가 필요한 정보를 한눈에 확인하기 어렵습니다.

1. **공고의 파편화**
   대형펌 채용 페이지, KICPA 구인 게시판, 일반 채용 사이트, 개별 회사 사이트에 공고가 분산되어 있습니다.
2. **CPA 조건의 불투명성**
   수습 CPA 가능 여부, 실무수습기관 인정 가능성, KICPA 우대/필수, 요구 연차, 마감일을 사용자가 직접 확인해야 합니다.
3. **마감일 중심 탐색의 어려움**
   오늘 마감, 이번 주 마감, 채용 시 마감, 상시채용 공고를 한 화면에서 비교하기 어렵습니다.
4. **지원 판단 정보의 분산**
   회사 규모, 평균연봉, 직원 수, 입사·퇴사 흐름, 외부 리뷰 링크 등 회사 정보가 공고와 분리되어 있습니다.
5. **기업회원의 채용 반응 파악 한계**
   플랫폼 내부 지원 기능이 없는 단계에서는 실제 지원자 수를 알 수 없기 때문에 조회, 북마크, 원문 클릭 같은 관심 지표가 필요합니다.

## Scope

Accountit은 **채용 공고 탐색과 큐레이션 경험**에 집중합니다.

- 공고 목록 조회 및 상세 조회
- CPA 특화 필터와 사용자별 빠른 필터
- 마감일 D-day 표시와 마감 임박 공고 강조
- 마감일 캘린더
- 원문 링크 연결
- 회사 공개 페이지
- 기업회원 공고 제출 및 관리자 승인
- 관리자 공고 등록, 수정, 마감 처리, 출처 관리
- AI 기반 공고 요약, 태그, 위험요소 검수
- 이력서 기반 공고별 AI 적합도 분석
- 관심 태그와 북마크 기반 알림
- 회계사 전용 커뮤니티와 CPA 인증 흐름

## Key Features

### 1. CPA 채용 공고 통합 탐색

KICPA 구인 게시판, 대형 회계법인 채용 페이지, 일반 채용 플랫폼, 개별 회사 채용 페이지의 공고를 하나의 리스트에서 탐색합니다.

![채용공고 화면](apps/web/public/hero/jobs-hero.png)

공고 카드에는 회사명, 공고명, 직무군, 마감일과 D-day, 고용 형태, 요구 경력, KICPA 조건, 수습 가능 여부, 실무수습기관 인정 가능성, 회사 유형, 출처, 최종 확인 시각을 함께 표시합니다.

### 2. CPA 맞춤형 필터

빠른 필터, 기본 필터, 상세 필터를 조합해 필요한 공고만 볼 수 있습니다.

| 구분 | 필터 |
| --- | --- |
| 빠른 필터 | 수습 CPA, 신입, 주니어 이직, 경력 이직, 마감 임박 |
| 기본 필터 | 직무군, 회사 유형, 지역 |
| 상세 필터 | 고용 형태, KICPA 조건, 수습 가능 여부, 실무수습기관 여부, 마감 유형, 경력, 업계 연봉 수준 |

### 3. AI 공고 요약 및 이력서 적합도 분석

공고 상세 페이지에서는 AI가 공고 핵심 내용을 요약하고 CPA에게 중요한 조건을 태그로 정리합니다. 개인회원은 이력서를 등록한 뒤 공고별 적합도 분석을 실행할 수 있습니다.

- 공고 핵심 요약
- 요구 경력과 자격 요약
- 주요 업무 요약
- KICPA 필수/우대, 수습 가능, 감사/세무/내부회계 등 추천 태그
- 계약직, 수습 불명확, 채용 시 마감 등 확인 필요 사항
- 이력서 대비 강점, 기업 우선순위, 보완점, 지원 추천 문구

### 4. 마감일 캘린더와 알림

주간·월간·타임라인 형태로 공고의 시작 일정과 마감 일정을 확인합니다.

![마감일 캘린더](apps/web/public/hero/calendar-hero.png)

- D-day 배지
- 오늘 마감, 이번 주 마감, D-7 마감 임박 표시
- 채용 시 마감 및 상시채용 별도 표시
- 북마크한 공고의 마감 임박 및 상태 변경 알림
- 관심 태그 기반 신규 공고 알림

### 5. 회사 공개 페이지와 회사 정보 시그널

공고와 연결된 회사 정보를 함께 제공해 지원 판단에 필요한 맥락을 줄입니다.

![회사 정보 화면](apps/web/public/hero/companies-hero.png)

- 회사 기본 정보와 회사 유형
- 현재 진행 중인 공고
- 회사 태그와 최근 업데이트 시각
- 평균연봉, 직원 수, 입사·퇴사 흐름 등 공개 데이터 기반 정보 표시 가능성
- 잡플래닛, 블라인드 등 외부 리뷰 링크

### 6. 기업회원 대시보드와 공고 성과 분석

기업회원은 공고 제출, 수정 요청, 회사 프로필 관리, 공고 성과 분석을 확인할 수 있습니다.

- 공고 조회수
- 원문 클릭수
- 북마크 수
- 원문 클릭률
- 북마크 전환율
- 일자별 관심 흐름
- AI 자동입력을 통한 공고 등록 폼 초안 생성

### 7. 회계사 전용 커뮤니티

CPA 준비생, 수습 회계사, 시니어 회계사, 자유게시판을 제공하고, 일부 게시판은 CPA 인증 상태를 기준으로 접근을 제한합니다.

![커뮤니티 화면](apps/web/public/hero/community-hero.png)

- 게시글 검색과 최신순/인기순 정렬
- Q&A 답변 및 채택
- 좋아요와 조회수
- 내가 쓴 글 모아보기
- CPA 인증 기반 수습 회계사 게시판 접근 제어

## User Flow

| 사용자 | 주요 흐름 |
| --- | --- |
| 방문자 / 개인회원 | 채용공고 탐색 → CPA 필터 적용 → 공고 상세 확인 → 원문에서 지원 |
| 개인회원 | 북마크 저장 → 마감/상태 변경 알림 확인 → 이력서 기반 AI 적합도 분석 |
| 개인회원 | 마감일 캘린더 확인 → 회사 공개 페이지 비교 → 커뮤니티 참여 |
| 기업회원 | 공고 게시 요청 → 관리자 검수 → 공고 게시 → 공고 성과 분석 |
| 관리자 | 공고/회사/출처 관리 → AI 제안 검수 → CPA 인증 요청 검수 |

## Tech Stack

- Monorepo: npm workspaces
- Web: Next.js App Router, TypeScript, Tailwind CSS, shadcn/ui, lucide-react
- API: NestJS, Prisma, PostgreSQL, Swagger/OpenAPI
- Shared: TypeScript enums and DTO-friendly types
- Auth: username/password, argon2 password hashing, HTTP-only JWT cookies
- Roles: `JOB_SEEKER`, `COMPANY`, `ADMIN`
